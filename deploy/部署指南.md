# CloudDisk Pro 部署指南

> 依赖栈一键启动：`docker compose -f deploy/docker-compose.yml up -d`  
> 数据库初始化：`mysql -uroot -proot < sql/init.sql`（含 `cloud_disk` + `xxl_job`）

---

## 一、Docker 依赖栈

| 服务 | 端口 | 说明 |
|------|------|------|
| MySQL | **3307**→3306 | 库名 `cloud_disk`，root/root |
| Redis | 6379 | 无密码 |
| MinIO | 9000 / 9001 | minioadmin / minioadmin |
| RabbitMQ | 5672 / 15672 | guest / guest |
| Elasticsearch | 9200 | 全文检索 |
| Kibana | 5601 | 日志可视化 |
| SkyWalking UI | 8188 | 链路追踪 |

**可选 profile**（按需追加）：

```bash
docker compose -f deploy/docker-compose.yml --profile xxl --profile sentinel --profile onlyoffice --profile monitoring --profile ldap up -d
```

| Profile | 服务 | 端口 |
|---------|------|------|
| `xxl` | XXL-JOB Admin | 8080（admin/123456） |
| `sentinel` | Sentinel Dashboard | 8858（sentinel/sentinel） |
| `onlyoffice` | Document Server | 8082 |
| `monitoring` | Spring Boot Admin | 8090 |
| `ldap` | OpenLDAP 测试 | 389 |

后端连接 Docker MySQL：`spring-boot.run.profiles=docker`

## 二、MinIO 对象存储

**Docker**（推荐）：compose 已含 MinIO，桶名 **`cloud-disk`**。

**本机安装**（Windows 示例）：

```bat
minio.exe server D:\minio\data --console-address ":9001"
```

| 项 | 值 |
|----|-----|
| API | http://127.0.0.1:9000 |
| 控制台 | http://127.0.0.1:9001 |
| 账号/密码 | minioadmin / minioadmin |

```yaml
clouddisk:
  storage:
    type: minio
  minio:
    endpoint: http://127.0.0.1:9000
    access-key: minioadmin
    secret-key: minioadmin
    bucket: cloud-disk
```

验证：`GET /api/storage/info`，上传后在控制台可见对象。

## 三、CDN 加速

路径：`用户 → CDN → MinIO`，通过 **预签名 URL + CDN 域名替换** 实现。

```yaml
clouddisk:
  cdn:
    enabled: true
    domain: https://cdn.example.com
```

| API | 说明 |
|-----|------|
| `GET /api/files/{id}/direct-url` | 登录用户直链 |
| `GET /share/{code}/direct-url?fileId=` | 分享直链 |

前端预览/下载优先走直链，失败回退后端代理。预签名默认 3600 秒有效。

## 四、ElasticSearch 全文检索

```bash
docker compose -f deploy/docker-compose.yml up -d elasticsearch
mvn spring-boot:run -Dspring-boot.run.profiles=local,es
```

```yaml
clouddisk.elasticsearch.enabled: true
spring.elasticsearch.uris: http://127.0.0.1:9200
```

- 镜像已内置 IK + 拼音插件（`deploy/elasticsearch/Dockerfile`），首次启动需 build：`docker compose -f deploy/docker-compose.yml build elasticsearch`
- 管理端：`POST /api/admin/search/rebuild` 重建索引

## 五、企业级扩展

### FFmpeg（视频转码）

安装 FFmpeg 并加入 PATH，默认异步处理：

- 封面 → `poster_path`
- H.264 MP4 → `transcode_path`（预览优先）
- 状态 → `transcode_status`

```yaml
clouddisk.ffmpeg.enabled: true
clouddisk.ffmpeg.path: ffmpeg
```

### RabbitMQ（异步媒体任务）

```bash
mvn spring-boot:run -Dspring-boot.run.profiles=local,mq
```

队列：`queue.thumbnail`、`queue.video-transcode`、`queue.notification`

### XXL-JOB（分布式定时）

```bash
docker compose -f deploy/docker-compose.yml --profile xxl up -d xxl-job-admin
mvn spring-boot:run -Dspring-boot.run.profiles=local,xxl
```

执行器 AppName：`clouddisk-pro`

| JobHandler | Cron 建议 |
|------------|-----------|
| `cleanExpiredSharesJob` | `0 0 * * * ?` |
| `cleanOldRecycleJob` | `0 30 2 * * ?` |
| `cleanExpiredChunksJob` | `0 15 3 * * ?` |

### Sentinel（限流）

默认开启。规则：

| 资源 | 默认 QPS |
|------|----------|
| `upload_api` / `simple_upload` | 30 |
| `auth_login` | 10 |
| `auth_register` | 5 |

应用层 Redis 限流（`clouddisk.rate-limit`）：登录/注册/分享 IP 限流、全站 API 300 次/分钟、上传 60 次/分钟、登录失败锁定与算术验证码。

生产务必使用 `deploy/nginx.conf.example` 中的 `limit_req` / `limit_conn`，且 **禁止公网直连 8088**，仅允许 Nginx 回源并设置 `X-Real-IP`。

Dashboard：

```bash
docker compose -f deploy/docker-compose.yml --profile sentinel up -d sentinel-dashboard
# http://127.0.0.1:8858
```

关闭：`clouddisk.sentinel.enabled: false`

### OnlyOffice（Office 在线编辑）

```bash
docker compose -f deploy/docker-compose.yml --profile onlyoffice up -d onlyoffice
mvn spring-boot:run -Dspring-boot.run.profiles=local,onlyoffice
```

- JWT 密钥需与 Document Server 一致
- `callback-base-url` 须为 Document Server 能访问的后端地址
- 全屏编辑：`/office/{fileId}`

### ClamAV（病毒扫描，可选）

```bash
mvn spring-boot:run -Dspring-boot.run.profiles=local,clamav
```

## 八、Nginx 生产部署

完整示例见 [nginx.conf.example](nginx.conf.example)，要点：

- 前端静态资源 + `/api/`、`/share/`、`/ws/` 反代后端 **8088**
- `client_max_body_size 20g`（与后端上传限制一致）
- WebSocket 需 `Upgrade` 头
- OnlyOffice 反代 **8082**；MinIO CDN 回源 **9000**
