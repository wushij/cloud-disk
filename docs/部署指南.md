# CloudDisk Pro 部署指南

> 依赖栈一键启动：`docker compose -f docs/docker-compose.yml up -d`
> 数据库初始化：`mysql -uroot -proot < sql/init.sql`

---

## 一、Docker 依赖栈

| 服务 | 端口 | 说明 |
|------|------|------|
| MySQL | 3306 | 库名 `CloudDisk`，root/root123 |
| Redis | 6379 | 无密码 |
| MinIO | 9000 / 9001 | minioadmin / minioadmin |
| RabbitMQ | 5672 / 15672 | guest / guest |
| Elasticsearch | 9200 | 全文检索（IK + 拼音插件） |

**可选 profile**（按需追加）：

```bash
docker compose -f docs/docker-compose.yml --profile elk --profile xxl --profile sentinel --profile onlyoffice --profile monitoring --profile ldap up -d
```

| Profile | 服务 | 端口 |
|---------|------|------|
| `elk` | Kibana + Logstash + SkyWalking | 5601 / 5044 / 8188 |
| `xxl` | XXL-JOB Admin | 8080（admin/123456） |
| `sentinel` | Sentinel Dashboard | 8858（sentinel/sentinel） |
| `onlyoffice` | Document Server | 8082 |
| `monitoring` | Spring Boot Admin | 8090 |
| `ldap` | OpenLDAP 测试 | 389 |

## 二、MinIO 对象存储

Docker compose 已含 MinIO，桶名 `cloud-disk`。

**本机安装**（Windows 示例）：

```bat
minio.exe server D:\minio\data --console-address ":9001"
```

| 项 | 值 |
|----|-----|
| API | http://127.0.0.1:9000 |
| 控制台 | http://127.0.0.1:9001 |
| 账号/密码 | minioadmin / minioadmin |

## 三、CDN 加速

路径：`用户 → CDN → MinIO`，通过预签名 URL + CDN 域名替换实现。

```yaml
clouddisk:
  cdn:
    enabled: true
    domain: https://cdn.example.com
```

| API | 说明 |
|-----|------|
| `GET /api/files/{id}/direct-url` | 登录用户直链 |
| `GET /share/{code}/direct-url?fileId=` | 分享直链 |

## 四、ElasticSearch 全文检索

ES 镜像已内置 IK + 拼音插件（`docs/elasticsearch/Dockerfile`），首次启动需 build：

```bash
docker compose -f docs/docker-compose.yml build elasticsearch
docker compose -f docs/docker-compose.yml up -d elasticsearch
```

管理端：`POST /api/admin/search/rebuild` 重建索引。

## 五、企业级扩展

### FFmpeg（视频转码）

安装 FFmpeg 并加入 PATH，配置：

```yaml
clouddisk.ffmpeg.enabled: true
clouddisk.ffmpeg.path: ffmpeg
```

### RabbitMQ（异步媒体任务）

队列：`queue.thumbnail`、`queue.video-transcode`、`queue.notification`

### XXL-JOB（分布式定时）

| JobHandler | Cron 建议 |
|------------|-----------|
| `cleanExpiredSharesJob` | `0 0 * * * ?` |
| `cleanOldRecycleJob` | `0 30 2 * * ?` |
| `cleanExpiredChunksJob` | `0 15 3 * * ?` |

### Sentinel（限流）

| 资源 | 默认 QPS |
|------|----------|
| `upload_api` / `simple_upload` | 30 |
| `auth_login` | 10 |
| `auth_register` | 5 |

应用层 Redis 限流：登录/注册/分享 IP 限流、全站 API 300 次/分钟。
生产务必使用 `docs/nginx.conf` 中的 `limit_req` / `limit_conn`，且**禁止公网直连 8088**。

### OnlyOffice（Office 在线编辑）

- JWT 密钥需与 Document Server 一致
- `callback-base-url` 须为 Document Server 能访问的后端地址

## 六、Nginx 生产部署

完整配置见 `docs/nginx.conf`，要点：

- UA 自动识别：PC → `pc/index.html`，手机 → `mobile/index.html`
- 安全 header：X-Frame-Options / X-Content-Type-Options / X-XSS-Protection
- gzip 压缩、静态资源 1 年缓存
- `client_max_body_size 20g`（与后端一致）
- WebSocket 需 `Upgrade` 头
- OnlyOffice 反代 8082；MinIO CDN 回源 9000
